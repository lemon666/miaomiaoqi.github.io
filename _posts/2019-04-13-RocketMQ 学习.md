# RocketMQ介绍

RcoketMQ 是阿里开源的消息中间件, 目前也已经孵化为 Apache 顶级项目, 它是纯 Java 开发, 具有**高吞吐量, 高可用性, 适合大规模分布式系统应用的特点**. RocketMQ 思路起源于 Kafka, 它对消息的可靠传输及事务性做了优化, 目前在阿里集团被广泛应用于交易, 充值, 流计算, 消息推送, 日志流式处理, binlog分发等场景.
